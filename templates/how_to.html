Правила ввода запросов

В поле поиска можно ввести максимум 3 элемента разных типов. Ввод с пробелами между элементами.

  1. Лексема
    Поиск выдаст все вхождения по лексеме. Пример: стали
  2. Точная форма
    Если ввести какую-либо форму в кавычках, найдутся все вхождения именно этой формы. Пример: "стали"
  3. Часть речи
    Поиск выдает все вхождения с указанной частью речи. Список доступных частей речи по ссылке ниже. Пример: ADJ
  4. Словоформа+частеречный тег
    Выдаются все вхождения, в которых данной форме соотвествует данная часть речи. Пример: стали+NOUN
